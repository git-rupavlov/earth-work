# Tomato Pruning

Knowledge base page for tomato pruning, suckers, single-stem / multi-stem choices, balcony constraints and propagation from cuttings.

## Practical scope

```text
crop: tomato
operation: pruning / training / propagation
plants: Pink Dream tomato, cherry tomato population
```

## Track here

- date of pruning;
- node position;
- removed sucker length;
- rooting attempt;
- wound healing;
- flowering impact;
- fruiting impact;
- plant height after pruning.
