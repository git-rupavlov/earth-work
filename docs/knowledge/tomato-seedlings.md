# Tomato Seedlings

Operational notes for tomato seedling starts, transplant timing, hardening, light, water and early stress signs.

## Scope

```text
crop: tomato
stage: seedling / young transplant
use: balcony and field planning
```

## Baseline checks

- emergence date;
- first true leaves;
- stem thickness;
- internode length;
- leaf color;
- watering interval;
- light exposure;
- transplant shock signs.
