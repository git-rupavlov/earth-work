# Project Overview

## Purpose

RP Project is a long-term regenerative agriculture initiative focused on practical soil improvement and low-input food production.

The project is designed around the following priorities:

1. Improve soil structure.
2. Increase organic matter.
3. Increase biological activity.
4. Improve water retention.
5. Produce biomass internally.
6. Produce food in limited but useful quantities.
7. Document observations and results.

## Scope

The initial working model assumes access to a greenhouse or former agricultural structure with surrounding land.

The first implementation phase is conservative by design.

The project does not assume full use of the available area during the first year.

## Initial Development Logic

```text
Assessment
    ↓
Small-scale cultivation
    ↓
Cover crops
    ↓
Biomass generation
    ↓
Compost and mulch
    ↓
Improved soil
    ↓
Expanded cultivation
```

## Non-Goals

The project is not intended to start as:

- commercial vegetable production;
- full-area greenhouse exploitation;
- high-input intensive farming;
- a system dependent on purchased fertilizers.

## Operating Model

The first stage is based on controlled experimentation and observation.

Each bed or growing zone should have a defined purpose:

- biomass production;
- nitrogen fixation;
- soil coverage;
- food production;
- microbial stimulation;
- nutrient mobilization.
