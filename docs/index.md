# RP Project

RP Project is a practical regenerative agriculture documentation project.

The project focuses on improving soil function through low-input systems, cover crops, biomass production, composting, mulch and observation.

## Primary Objective

The primary objective is not maximum short-term yield.

The primary objective is to establish a system where soil quality improves each season through:

- living roots;
- permanent soil cover;
- internal biomass production;
- composting;
- reduced disturbance;
- observation and documentation.

## Working Principle

```text
Soil improvement
      ↓
Biological activity
      ↓
Water retention
      ↓
Plant health
      ↓
Food production
```

Food production is treated as a consequence of soil development, not as the only success metric.
