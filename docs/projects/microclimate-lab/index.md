# Microclimate Lab

## Status

Planned / active development.

## Purpose

Microclimate Lab documents indoor and small-scale environmental monitoring.

The goal is to measure and compare:

- temperature;
- humidity;
- airflow;
- light exposure;
- shelf-level differences;
- greenhouse and balcony microclimates.

## Feeds Data Into

- Sprout Lab;
- Balcony Pot House;
- Greenhouse 2026;
- Monitoring & Automation;
- Field Journal.

## Candidate Hardware

- ESP32;
- Raspberry Pi Pico;
- SHT31 / SHT41;
- BMP280;
- light sensor;
- low-speed PC fans;
- local dashboard or simple log files.

## First Questions

- How different are upper and lower shelf temperatures?
- How fast does humidity build up under covers?
- How much airflow is enough to reduce mold risk?
- How much light reaches each shelf?
