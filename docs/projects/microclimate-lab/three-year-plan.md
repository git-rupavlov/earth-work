# Microclimate Lab - 3-Year Development Plan

## Purpose

Build a practical monitoring layer for indoor and small-scale growing systems.

## Year 1

- measure temperature and humidity;
- compare shelf-level conditions;
- test airflow effects;
- select reliable sensors;
- start simple logs.

## Year 2

- connect data to Sprout Lab and Balcony Pot House;
- improve sensor placement;
- test dashboards;
- document useful thresholds.

## Year 3

- reuse the monitoring pattern for greenhouse and field projects;
- reduce manual checking;
- keep only sensors that produce useful decisions.
