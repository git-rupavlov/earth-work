# Microclimate Lab - Site Assessment

## Scope

Microclimate Lab assesses conditions inside small growing environments.

## Assessment Focus

- temperature;
- humidity;
- light exposure;
- airflow;
- sensor placement;
- data reliability.

## Locations Covered

- Sprout Lab;
- Balcony Pot House;
- Greenhouse 2026;
- future indoor systems.

## Key Questions

- Where are the hot spots?
- Where does humidity build up?
- Which shelves receive less light?
- Which measurements actually change decisions?

## Next Actions

- test upper and lower shelf sensors;
- record daily minimum and maximum values;
- compare passive and fan-assisted airflow.
