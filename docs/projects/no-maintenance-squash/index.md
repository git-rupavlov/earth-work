# No-Maintenance Squash Trial

## Status

Active / seasonal.

## Purpose

This project tests how much useful food can be produced with minimal intervention.

The focus is not neglect. The focus is measuring which crops tolerate low input conditions while still producing something worth repeating.

## Main Questions

- How much water is required?
- How much labor is required?
- How much yield is produced?
- How well does the crop suppress weeds?
- How well does the harvest store?

## Feeds Data Into

- Crop Matrix;
- Field Journal;
- Plant Profiles;
- 3-Year Plan.
