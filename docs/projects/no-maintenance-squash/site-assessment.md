# No-Maintenance Squash Trial - Site Assessment

## Location

Multiple locations using scattered pits.

## Assessment Focus

- pit location;
- soil moisture;
- shade and sun exposure;
- mulch availability;
- nearby water access;
- weed competition;
- pest and wildlife pressure.

## Key Questions

- Which pits survive with the least help?
- Which locations hold moisture longest?
- Which mulch setup performs best?
- How much yield is produced per visit and per watering event?

## Next Actions

- record each pit location;
- photograph each pit;
- track watering events;
- record survival and yield.
