# Greenhouse 2026 - Site Assessment

## Location

Protected cultivation site.

## Current State

The greenhouse currently contains mostly lapad and nettle, with a smaller amount of vegetables.

## Assessment Focus

- existing perennial biomass;
- soil moisture;
- ventilation;
- heat buildup;
- water access;
- internal workflow;
- seedling and seed production zones.

## Key Questions

- Which existing plants are useful and should remain?
- Where is heat stress highest?
- Where can seedlings or amaranth seed trials fit without disrupting the system?
- How much biomass can be harvested and cycled back into compost or mulch?

## Next Actions

- map existing vegetation;
- define protected trial zones;
- record temperature and watering needs;
- document harvest and biomass cycles.
