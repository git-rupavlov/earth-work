# Crop Oracle

Decision-support page for crop selection and planning.

## Purpose

The Crop Oracle should compare crops by:

- nutrition value;
- mineral density;
- useful compounds;
- water demand;
- care demand;
- required area;
- sugar, protein and fat profile;
- yield per square metre;
- balcony / field fit;
- resilience.

For now, it is the future home of crop recommendation logic. A crystal ball, but with YAML instead of theatrical nonsense.
