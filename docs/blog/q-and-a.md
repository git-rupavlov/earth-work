# Q&A

Collected questions and answers from the project.

## Purpose

This page is for short-form Q&A that should not become full knowledge-base articles yet.

Examples:

- plant symptoms;
- quick comparisons;
- planting timing;
- container questions;
- source notes;
- field decisions.

A staging area for useful chaos, because pretending all notes are polished articles is how wikis go to die.
